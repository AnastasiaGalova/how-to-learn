# Научиться учиться - первый проект в профессии веб-разработчика

Первый самостоятельный проект в рамках обучения в Яндекс Практикуме по курсу "веб-разработчик". Проект написан с помощью методологии БЭМ, файловая структура проекта создана по правилам Nested БЭМ. Проект представляет из себя одностраничный сайт с полезной информацией для обучения. В проекте использована флексбокс-вёрстка, анимации, iframe.  

## Планы по доработке
- Сделать сайт кроссбраузерным и дописать все вендорные префиксы.
- Задизайнить форму, через которую пользователи смогут отправлять комментарии.
